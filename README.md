### Getting Started

1. Clone this repository

```bash
git clone https://github.com/palthol/portfolio-app.git
cd portfolio-app
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Visit `http://localhost:5173` in your browser
